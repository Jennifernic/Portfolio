------Portfolio Website---------

# Project Overview:
This is a personal portfolio website created to showcase my skills, projects, and contact details. The website includes sections such as Home, About, Skills, Projects, and Contact to help potential employers, clients, or collaborators get to know more about my work and experience.

# Features:
- Home Section: Introduction with a brief description and dynamic text displaying different skills.
- About Section: A personal introduction with details about my background and skills.
- Skills Section: Displays technical skills with progress bars or radial bars showing proficiency.
- Projects Section: Showcases different projects I've worked on with descriptions and links.
- Contact Section: Provides a contact form that allows visitors to send messages directly to my email.

# Technologies Used:
- HTML5: For the basic structure and layout of the website.
- CSS3: For styling the elements and making the website visually appealing.
- JavaScript: For adding interactivity, including form submission and dynamic text animation.
- EmailJS: To enable direct email communication from the contact form.

# How to Use:
Clone or Download the Repository:

Clone the repository to your local machine using git clone https://github.com/Jennifernic/Portfolio.git
Or download the ZIP file and extract it.

Open the Website:
Open index.html in a web browser to view the portfolio.
